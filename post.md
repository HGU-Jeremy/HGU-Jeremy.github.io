# 마크다운 문법 5개 이상 사용 연습

## 1. 코드블럭
(```)을 이용하면 코드블럭을 사용가능 하다.

```
printf("hello world\n");
```

## 2. 링크
```
[link keyword][id]

[id]: URL "Optional Title here"

// code
Link: [Google][googlelink]

[googlelink]: https://google.com "Go google"
```
Link: [Google][googlelink]

[googlelink]: https://google.com "Go google"

## 3. 헤더
글머리는 6개까지 사용가능
```
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6
```
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6

## 4. BlockQuote
이메일에서 사용하는 > 블럭인용문자를 이용한다.

```
> This is a first blockqute.
>	> This is a second blockqute.
>	>	> This is a third blockqute.
```
> This is a first blockqute.
>	> This is a second blockqute.
>	>	> This is a third blockqute.

## 5. 목록
순서없는 목록(글머리 기호: *, +, - 지원)
```
* 빨강
  * 녹색
    * 파랑

+ 빨강
  + 녹색
    + 파랑

- 빨강
  - 녹색
    - 파랑
```
